# JARVIS - Python Voice Assistant

## Overview

JARVIS is a simple voice-controlled virtual assistant built using Core Python. It can listen to user commands, recognize speech, and perform various tasks such as opening websites through Google search and responding to basic voice commands.

This project was created to explore speech recognition, voice interaction, and Python automation concepts.

---

## Features

* 🎤 Voice command recognition
* 🔊 Text-to-speech responses
* 🌐 Open websites using Google Search
* ⏰ Basic assistant functionalities
* 🖥️ Hands-free interaction through voice commands
* ⚡ Lightweight and easy to use

---

## Technologies Used

* Python
* SpeechRecognition
* pyttsx3
* webbrowser
* datetime
* os

---

## How It Works

1. The assistant listens for voice commands using the microphone.
2. Speech is converted into text using Python's speech recognition libraries.
3. The command is processed.
4. Based on the command, JARVIS performs actions such as:

   * Opening websites
   * Searching Google
   * Providing spoken responses

## Installation

### Clone the Repository

```bash
git clone https://github.com/Nikhil-Khanna-be/Jarvis-Voice-Chatbot.git
cd Jarvis-Voice-Chatbot
```

### Install Dependencies

```bash
pip install SpeechRecognition pyttsx3 pyaudio
```

## Limitations

* This project does not include ChatGPT or any AI-based conversational model.
* ChatGPT integration was not implemented because it requires access to a paid API service.
* The assistant currently supports a limited set of predefined commands.
* Internet connection is required for website searches and opening web pages.
